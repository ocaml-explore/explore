## Overview 

OCamlfind is a command-line tool for using the findlib library management system. The main OCaml website has a thorough [tutorial](https://ocaml.org/learn/tutorials/compiling_ocaml_projects.html) on using the tool as it used to be the defacto standard for building OCaml projects.
